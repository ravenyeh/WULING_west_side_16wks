# 西進武嶺 SUB4 16週訓練計劃

一個專為挑戰西進武嶺 SUB4（4小時內完成）設計的訓練計劃網站，整合 Garmin Connect API，支援個人化功率訓練目標。

## 賽事資訊

- **路線**：埔里地理中心碑 → 武嶺
- **總距離**：54 公里
- **總爬升**：2,000+ 公尺
- **最高點**：海拔 3,275 公尺（台灣公路最高點）
- **目標時間**：3小時50分（SUB4）

## 功能特色

### 個人化設定
- **比賽日期**：設定您的目標賽事日期，自動計算倒數計時
- **FTP 輸入**：輸入您的功能性閾值功率（FTP），自動生成個人化功率目標

### 16週訓練計劃
完整的 112 天訓練課表，分為四個訓練週期：

| 週期 | 週次 | 重點 |
|------|------|------|
| 基礎期 | Week 1-4 | 建立有氧基礎，提升基礎體能 |
| 建構期 | Week 5-8 | 提升閾值功率，強化爬坡能力 |
| 巔峰期 | Week 9-12 | 最大訓練量，模擬賽事強度 |
| 減量期 | Week 13-16 | 恢復體能，迎接比賽 |

### 7區段配速策略

| 區段 | 路線 | 距離 | 爬升 | 目標功率 |
|------|------|------|------|----------|
| 1 | 埔里 → 人止關 | 14km | 250m | FTP 65-70% |
| 2 | 人止關 → 霧社 | 10km | 450m | FTP 70-75% |
| 3 | 霧社 → 清境 | 8km | 600m | FTP 70-78% |
| 4 | 清境 → 翠峰 | 9km | 560m | FTP 65-72% |
| 5 | 翠峰 → 鳶峰 | 6km | 450m | FTP 65-70% |
| 6 | 鳶峰 → 昆陽 | 3km | 335m | FTP 65-72% |
| 7 | 昆陽 → 武嶺 | 4km | 185m | FTP 70-85% |

### 動態功率目標
當您輸入 FTP 後，系統會自動：
- 將所有 FTP 百分比轉換為實際瓦數
- 顯示個人化的 6 區功率區間（Coggan Zones）
- 在訓練表格中顯示目標功率範圍

### Garmin Connect 整合
- 一鍵匯入訓練至 Garmin Connect
- 自動排程至指定日期
- 支援下載 JSON 格式訓練檔

## 使用說明

1. **設定比賽日期**：在頁面頂部輸入您的目標賽事日期
2. **輸入 FTP**：輸入您的功能性閾值功率（建議範圍 100-500W）
3. **儲存設定**：點擊「儲存設定」按鈕
4. **查看訓練**：瀏覽每日訓練內容，點擊「查看」按鈕獲取詳細資訊
5. **匯入 Garmin**：在訓練詳情中輸入 Garmin 帳號密碼，一鍵匯入

## 功率區間說明（Coggan Zones）

| 區間 | 名稱 | FTP 百分比 | 用途 |
|------|------|------------|------|
| Z1 | 恢復 | < 55% | 積極恢復 |
| Z2 | 耐力 | 55-75% | 有氧基礎 |
| Z3 | 節奏 | 75-90% | 節奏騎乘 |
| Z4 | 閾值 | 90-105% | 乳酸閾值 |
| Z5 | VO2max | 105-120% | 最大攝氧量 |
| Z6 | 無氧 | > 120% | 無氧爆發 |

## SUB4 關鍵要點

1. **功率管理**：前半段保守配速，控制在 FTP 65-75%
2. **補給策略**：每 20 分鐘補給一次，全程攝取 60-90g 碳水/小時
3. **高海拔適應**：海拔 2500m 以上功率下降 10-15% 是正常的
4. **心理準備**：分段目標管理，專注當下區段
